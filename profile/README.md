**HopDuck** is a modern SaaS platform blending hyper-secure URL shorteners with dynamic, device-aware deep links, geo targeting rules, and sub-second edge routing.

Paste a URL in. Get a short one out. Done.

## What it does

- Shorten long URLs into clean, shareable links
- Track clicks and basic analytics
- Custom aliases (coming soon)
- QR code generation (coming soon)

## Stack

| Layer | Tech |
|---|---|
| Frontend & API | [Next.js](https://nextjs.org) |
| Database | [PostgreSQL](https://www.postgresql.org) via [Neon](https://neon.tech) |
| ORM & Migrations | [Drizzle Kit](https://orm.drizzle.team) |
| Caching | [Redis](https://redis.io) (coming soon) |
| Hosting | [Vercel](https://vercel.com) |
